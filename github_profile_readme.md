## 🚀 Key Projects

### 💧 [AquaSense: Smart Soil Moisture Irrigation System](https://github.com/muhammadbilalwaris/aquasense-smart-irrigation)
A closed-loop automated watering controller for ESP32. Implements ADC linear calibration, threshold hysteresis protection, max run timers, cooldown lockouts, and features a full Python CLI physical simulator.

### 🧪 [CallibraSense: Automated Sensor Calibration System](https://github.com/muhammadbilalwaris/calibrasense)
An automated sensor calibration toolkit. It interfaces with sensors via serial communication, processes data in Python, compares against reference baselines, and calculates calibration parameters.

### 📊 [Automated Data-Logging System](https://github.com/muhammadbilalwaris/Automated-Data-Logging-System)
A hybrid hardware-software data acquisition node using an ESP32, 16-bit ADC, OLED display, and microSD logging, paired with a custom Python GUI dashboard.

### ⚡ [Automatic Phase Selector for Three-Phase Supply](https://github.com/muhammadbilalwaris/Automatic-phase-selector-for-three-phase-supply)
An ESP32-based industrial voltage monitoring system that performs phase fault detection, automatic relay routing, and includes a Python simulator.
